# farmer-to-consumer
The Farmer to Consumer Direct Marketplace System is a web-based platform that connects farmers directly with consumers without involving middlemen. In the traditional system, farmers sell products at low prices to intermediaries, while consumers buy them at higher prices. This system allows farmers to upload product details, set prices, and manage orders online. Consumers can view available products, compare prices, and purchase directly from farmers. The main goal of this system is to increase farmers’ profit, reduce product cost for consumers, and create a transparent and efficient agricultural marketplace.
